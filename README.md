# Tutorial de configuracion del proyecto
## 1. Clonar el repositorio:

git clone https://github.com/rafaferne/SmartMedTFG.git

## 2. Entrar en la carpeta del proyecto y ejecutar Docker (importante tener Docker ejecutando)

cd SmartMedTFG
docker compose up -d

## 3. Acceder a la pagina usando la URL

Poner este link en el buscador y listo: [text](http://localhost/)
